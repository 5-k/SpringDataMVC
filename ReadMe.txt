Backend: Spring MVC, Security, Hibernate
FrontEnd: AnguarJS 1.6, HTML, CSS, Javascript

SQL Script Path: src/main/resources
Update mysql Details: src/main/resources/application.properties
Softwares needed to run: Maven, Java

Prior running the application, run the below commands from the folder where you have extracted the ZIP File. 
          1. mvn eclipse:eclipse
          2. mvn clean install

Run command: mvn spring-boot:run
